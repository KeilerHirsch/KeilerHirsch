Problem → thesis → buildable solution, written up in EN/DE from months of hands-on iteration, not theory.

**Security research:** I find and responsibly disclose issues in the AI tooling I run daily — permission/trust-boundary bugs in agent CLIs, not just report-and-forget. Coordinated disclosure only, no public details before a fix ships.
